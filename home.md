# Vamp — Mobile App Ecosystem

**Developer:** Vamp

---

## About

Vamp is an ecosystem of Android applications designed for everyday use — file management, communication, security, search, and more. All apps work independently and can interact with each other through a shared data bus.

## Applications

| App | Purpose |
|-----|---------|
| **Haron** | File manager |
| **Beresta** | Messenger |
| **Morok** | Security & antivirus |
| **Hora** | Clock & alarms |
| **Iris** | Camera |
| **Hermes** | Search |
| **Divo** | Wallpapers |
| **GdeDeti** | Parental control |
| **Mara** | Medical catalog |
