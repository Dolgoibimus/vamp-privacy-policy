## Русский

### Область действия

Настоящая политика конфиденциальности распространяется на все приложения, опубликованные разработчиком **Vamp**.

### Сбор данных

Приложения **не собирают, не хранят на серверах и не передают третьим лицам** никакие личные данные пользователей. Все данные (настройки, файлы, история) хранятся **локально на устройстве** пользователя.

### Разрешения устройства

Приложения могут запрашивать следующие разрешения в зависимости от функциональности:

- **Доступ к файлам** — для просмотра, копирования, перемещения и управления файлами на устройстве
- **Камера** — для сканирования QR-кодов, штрихкодов и съёмки фото
- **Микрофон** — для голосовых команд и голосовой связи
- **Сеть (Wi-Fi, Интернет)** — для передачи файлов между устройствами, подключения к облачным хранилищам и удалённым серверам
- **Bluetooth** — для передачи файлов и удалённого управления
- **USB** — для работы с внешними USB-накопителями
- **Уведомления** — для отображения прогресса фоновых операций
- **Биометрия** — для защиты приложения (PIN, отпечаток пальца)
- **Foreground Service** — для выполнения длительных операций (копирование, передача файлов)

Каждое разрешение используется **исключительно** для заявленной функции. Данные, полученные через эти разрешения, не покидают устройство пользователя.

### Облачные сервисы

Некоторые приложения позволяют подключаться к сторонним облачным сервисам (например, Google Drive, Dropbox, Яндекс Диск). При авторизации:
- Используется протокол **OAuth 2.0** — приложение не видит и не хранит пароль пользователя
- Токены доступа хранятся **локально на устройстве** в зашифрованном виде
- Токены **не передаются** разработчику или третьим лицам
- Пользователь может отключить аккаунт в любой момент — токены удаляются

### Сторонние сервисы

Приложения могут взаимодействовать со сторонними сервисами только по прямому запросу пользователя (облачные хранилища, трансляция на ТВ и т.д.). Политики конфиденциальности этих сервисов применяются отдельно.

Приложения **не содержат** рекламы, аналитики, трекеров или SDK для сбора данных.

### Безопасность

- Конфиденциальные данные (токены, пароли) шифруются на устройстве
- Приложения не используют собственные серверы для передачи или хранения пользовательских данных

### Дети

Приложения не предназначены для детей младше 13 лет и не собирают данные детей осознанно.

### Изменения

Мы оставляем за собой право обновлять данную политику. При существенных изменениях обновлённая версия будет опубликована по тому же адресу с новой датой.

## English

### Scope

This privacy policy applies to all applications published by **Vamp**.

### Data Collection

Applications **do not collect, store on servers, or share with third parties** any personal user data. All data (settings, files, history) is stored **locally on the user's device**.

### Device Permissions

Applications may request the following permissions depending on functionality:

- **File access** — to browse, copy, move, and manage files on the device
- **Camera** — to scan QR codes, barcodes, and take photos
- **Microphone** — for voice commands and voice communication
- **Network (Wi-Fi, Internet)** — for file transfer between devices, cloud storage access, and remote server connections
- **Bluetooth** — for file transfer and remote control
- **USB** — for external USB drive access
- **Notifications** — to display background operation progress
- **Biometrics** — for app protection (PIN, fingerprint)
- **Foreground Service** — for long-running operations (copying, file transfer)

Each permission is used **exclusively** for its stated purpose. Data obtained through these permissions does not leave the user's device.

### Cloud Services

Some applications allow connecting to third-party cloud services (e.g., Google Drive, Dropbox, Yandex Disk). During authorization:
- **OAuth 2.0** protocol is used — the app never sees or stores the user's password
- Access tokens are stored **locally on the device** in encrypted form
- Tokens are **never transmitted** to the developer or third parties
- Users can disconnect accounts at any time — tokens are deleted

### Third-Party Services

Applications may interact with third-party services only at the user's direct request (cloud storage, TV casting, etc.). These services' own privacy policies apply separately.

Applications **do not contain** advertisements, analytics, trackers, or data collection SDKs.

### Security

- Sensitive data (tokens, passwords) is encrypted on-device
- Applications do not use proprietary servers for transferring or storing user data

### Children

Applications are not intended for children under 13 and do not knowingly collect children's data.

### Changes

We reserve the right to update this policy. In case of significant changes, the updated version will be published at the same address with a new date.
